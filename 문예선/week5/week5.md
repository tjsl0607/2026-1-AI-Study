## Denoising Diffusion Probabilistic Models
### 개념 정리
- Diffusion Process (확산 과정)
원본 데이터에 아주 작은 가우시안 노이즈를 여러 단계(보통 1000번)에 걸쳐 점진적으로 추가하는 과정. 이 과정을 끝까지 거치면 원본 이미지는 형체를 알아볼 수 없는 완전한 노이즈가 됨.

- Denoising Process (역과정)
완전한 노이즈 상태에서 출발하여, 점진적으로 노이즈를 제거해 나가면서 원래의 선명한 이미지를 복원(생성)하는 과정. 인공지능 학습방법 핵심 = 노이즈를 걷어내는 방법

- 마르코프 체인
현재의 상태가 오직 바로 직전의 상태에만 영향을 받는 확률적 성질. DDPM은 노이즈를 추가하거나 뺄 때, 여러 단계 전을 보지 않고 바로 이전 단계의 데이터만을 참조하여 계산을 단순화함.

### 문제 제기
- 기존 이미지 생성 AI 모델들의 뚜렷한 한계 - 품질 vs 안정성 vs 다양성의 딜레마

#### GAN - Generative Adversarial Network
장: 매우 선명하고 실제 같은 고품질 이미지 생성. 
단: 학습이 극도로 불안정. 모드 붕괴 현상 - 모델이 특정한 형태의 이미지만 반복해서 만들어냄. 즉, 다양성 떨어짐.

#### VAE - Variational Autoencoder
장: 수학적으로 안정적 학습 가능 
다양한 이미지 생성
단: 생성된 결과물이 전체적으로 흐릿하고 선명도 떨어짐.
원인: 안정적인 확률 모델(수학적으로 계산하기 쉬운 구조), 고품질의 디테일 생성을 동시에 달성하는 최적화 방법이 부재

### 핵심 해결책
기존 방식: 복잡한 확률 분포나 원본 이미지를 직접 예측
- 노이즈가 낀 이미지에서 원본 이미지가 무엇일지 한 번에 예측
- 복잡한 확률 수식 전체 계산

#### DDPM - 단순한 노이즈 예측으로의 전환
신경망(U-Net 아키텍처 사용)이 원본 이미지를 예측 -> 현재 이미지에 껴있는 노이즈가 무엇인지만 예측하도록 학습

- 목적 함수의 단순화
복잡한 수학적 식(ELBO 등)을 전개 -> 불필요한 항들을 과감히 버리기 + 모델이 예측한 노이즈와 실제 정답 노이즈 간의 차이만을 줄이기로 매우 단순화

### 효과
1. 학습의 극적인 안정화
두 신경망이 싸우는 적대적 학습 (GAN) -> 정답 노이즈와 예측 노이즈의 오차만 줄이면 됨
-> 최적화와 학습 과정이 VAE처럼 매우 안정적

2. 고품질과 다양성의 동시 달성
단순화된 목적 함수 -> 모델이 이미지의 세세한 디테일을 학습하는 데 집중하도록 만듦. 
흐릿해지는 VAE의 단점과 다양성을 잃는 GAN의 단점을 모두 극복.

3. 결과 및 성과
고품질 이미지 생성 입증
CIFAR-10, LSUN 등의 표준 데이터셋에서 기존 SOTA(State-of-the-Art)였던 GAN과 필적하거나 이를 뛰어넘는 최고 수준의 이미지 생성 품질을 기록함.

=> 의의!! 생성형 AI의 새로운 패러다임 창조함. 확산 모델이 실질적으로 작동하며 압도적인 성능을 낼 수 있음을 증명함. 이후 등장한 DALL-E 2, Midjourney, Stable Diffusion 등 현재 세계를 주도하는 고성능 이미지/영상 생성 AI들의 핵심 기반 기술이 됨.
